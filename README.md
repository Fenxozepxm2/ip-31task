## Установка
1. Скачайте репозиторий : git clone __LINK__
2. Создайте вирутуальную среду
3. Активируйте виртуальную среду с помощью команды env\Scripts\activate
4. Установите зависимости: pip install django pillow
5. Запустите сервер: python manage.py runserver
